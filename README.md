# Projects 

This repository contains a collection of projects I made in my spare time to showcase my skills. They cover backend development, distributed systems, infrastructure, and machine learning.

Each project is self-contained and focuses on a specific area of software engineering. You can run them on most devices since they're adaptable. They have a few requirements, so see the individual readmes for more details. 

---

## Projects

### backend-api
A production-style backend API built with FastAPI.

**Highlights:**
- JWT authentication and RBAC
- CRUD operations with filtering and pagination
- Redis caching and rate limiting
- Clean, modular architecture

---

### task-queue
A distributed task queue system for asynchronous job processing.

**Highlights:**
- Redis-based job queue
- Multithreaded worker pool
- Retry logic and dead-letter queue
- Producer/consumer architecture

---

### ml-text-api
A machine learning API for text classification.

**Highlights:**
- TF-IDF + Logistic Regression model
- Real-time and batch prediction endpoints
- FastAPI-based deployment
- Lightweight and efficient inference

---

### infra-stack
A containerised infrastructure setup using Docker.

**Highlights:**
- Docker Compose orchestration
- Nginx reverse proxy
- Monitoring with Prometheus and Grafana
- Logging with Loki
